# ZombieArena

Zombie Arena Game

```
git clone https://github.com/slier81/ZombieArena
cd ZombieArena
mkdir build && cd build
cmake ..
make 
```
The game will be available inside `build` directory  

`w` - go up  
`s` - go down  
`a` - go left  
`d` - go right  
`r` - reload  
`space` - shoot
